# 지옥도 (jiokdo) — 목적 및 요구사항 정의서

**버전** 0.1 · **작성일** 2026-05-25 · **저장소** `sayaindecay/jiokdo`

---

## 1. 목적

비동기 텍스트 기반 TRPG 진행을 위한 **게시판형 굴림판**.
글과 댓글 본문 안에서 직접 다이스를 굴리고, 그 결과를 글에 영구히 새겨 사후 조작이 불가능한 신뢰 가능한 플레이 공간을 제공한다.

**문제 인식**
- 실시간 채팅 도구는 로그 보관·검색이 약하고 비동기 플레이에 부적합
- 외부 다이스 봇은 결과가 채팅에 흘러가며 게시판과 분리됨
- VTT(Roll20 등)는 텍스트 RP에 과한 무게감

**해결 방식**
- 게시판 글/댓글 본문에 명령어를 끼워 넣으면 서버가 굴려 결과를 함께 저장
- 한 번 저장된 굴림 결과는 변경 불가 (조작 불가능)
- Call of Cthulhu의 1d100 판정을 1차 타겟으로 자동 단계 분류

## 2. 타겟 사용자

| 페르소나 | 행동 | 주요 기능 |
|---|---|---|
| 키퍼/마스터 | 시나리오·세션 운영 | 글 작성, 판정 요청, NPC 굴림 |
| 플레이어 | 캐릭터 연기 + 굴림 | 댓글 작성, 능력 판정 |
| 관찰자 | 진행 로그 열람 | 글/댓글 조회 |

1차 타겟은 CoC 플레이어. 향후 D&D / 일반 NdM 시스템 확장.

## 3. 기능 요구사항 (FR)

### 3.1 게시판
- **FR-01** 사전 정의 게시판 3종 (`로비`/`시나리오`/`세션`) 자동 시드
- **FR-02** 홈에서 카드형 게시판 목록 표시
- **FR-03** 게시판 페이지에서 글 목록(최신순) + 댓글 수 표시

### 3.2 글
- **FR-04** 가입 없이 닉네임만으로 글 작성 가능 (미입력 시 "익명의 탐사자")
- **FR-05** 글: 제목 ≤ 120자, 본문 ≤ 20,000자
- **FR-06** 닉네임은 `localStorage`에 보존, 재방문 시 자동 채움
- **FR-07** 글 상세에서 작성자/시각/본문/댓글 표시

### 3.3 댓글
- **FR-08** 글 상세에서 댓글 작성 (닉네임 + 본문 ≤ 8,000자)
- **FR-09** 댓글에서도 다이스 명령 사용 가능
- **FR-10** 댓글 목록은 시간 오름차순

### 3.4 다이스
- **FR-11** 본문 작성 시 명령은 한 줄 단위로 자동 파싱
- **FR-12** 명령 종류
  - `/roll NdM[±K]` / `/r NdM[±K]` — 일반 굴림
  - `/cc [이름] 기능치` — CoC 1d100 판정
- **FR-13** 굴림은 **서버에서** 실행, 결과는 작성 시점 그대로 동결 저장
- **FR-14** 굴림 결과는 본문의 원래 줄 위치에 시각적 블록으로 렌더링
- **FR-15** 입력 한계: 주사위 개수 ≤ 100, 면 수 ≤ 1,000

### 3.5 CoC 판정 자동 분류
- **FR-16** 1d100 결과와 기능치를 비교해 6단계로 라벨링

| 단계 | 조건 |
|---|---|
| 결정적 성공 | 굴림 = 1 |
| 극단적 성공 | 굴림 ≤ ⌊기능치/5⌋ |
| 어려운 성공 | 굴림 ≤ ⌊기능치/2⌋ |
| 일반 성공 | 굴림 ≤ 기능치 |
| 실패 | 굴림 > 기능치 |
| 펌블 | 기능치<50 → 96–100 / 기능치≥50 → 100 |

### 3.6 도움말
- **FR-17** `/help` 페이지에 명령 문법·판정 규칙 표 제공

## 6. 데이터 모델

```sql
boards   (slug PK, name, description)
posts    (id PK, board_slug FK, nickname, title, segments_json, created_at)
comments (id PK, post_id FK→posts, nickname, segments_json, created_at)
```

본문은 `Segment[]` JSON으로 저장하여 텍스트와 굴림 결과를 같은 순서로 동결.
```ts
type Segment =
  | { type: "text"; value: string }
  | { type: "dice"; result: RollResult | CocResult };
```

## 10. 성공 지표 (KPI)

- 활성 게시판 수 ≥ 5
- 누적 글 ≥ 100
- 누적 굴림 ≥ 1,000
- 글당 평균 댓글 ≥ 3 (= 비동기 세션이 실제로 진행되고 있다는 신호)
