화면 2. 내 캠페인 · 로그인 후
주요 변경점
  • 이어서 플레이 카드를 가장 위 — 한 탭으로 즉시 본격 플레이로 진입
  • 내 캐릭터는 가로 스크롤 캐러셀 (한 화면에 1.5장 보임 — 추가 있다는 신호)
  • 데스크탑의 표(CampaignsTable) → 카드 리스트. 좌측 색 띠 = 캠페인 hue
  • 상태 필터 pill은 가로 스크롤 허용 (전부 / 활성 / 휴면 / 종료)
  • FAB(➕) = 새 캠페인 · 초대 코드 입력 (기존 collapsible 폼은 시트로 슬라이드업)
  • 하단 탭바 「캠페인」 활성 상태

다음 단계 · 후속 시안 후보

두 화면이 합의되면 우선 작업 권장 순서:

  1. 플레이 · 장면 모드 — 시트 bottom-sheet, 컴포저 sticky, PC 칩 가로 스크롤
  2. 플레이 · 전투 트래커 — 추천안: 세로 스택 + 이니셔티브 접이식
    (이니셔티브를 상단 collapsed bar로 두고, 펼치면 풀-체크리스트가 슬라이드 다운. 스탯블록 → 액션바 → 로그 순서로 자연스러운 위→아래 흐름. 탭 전환보다 가독성·동시 노출이 좋음.)
  3. 내 캐릭터 시트 — 섹션 아코디언 (특성치 / 파생 / 기능 / 무기 / 인벤토리). 굴림 버튼 = 큰 터치 타겟
  4. vital 편집 시트 — 화면 하단에서 슬라이드업 (현재의 인라인 편집기를 모달 시트로 격상)
  5. 에너미 작성 — 단일 컬럼 폼, 섹션별 collapsible 카드, 자동계산 패널은 항상 sticky 상단 노출
